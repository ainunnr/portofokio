SI-GAP

Prediabetes Screening Application (2021)

Description

This application was developed to help the first stage of prediabetes research. The data obtained from this application is displayed on the website for further analysis by researchers. Then each research participant will get counseling and education based on their individual analysis.

Features

  1. Google Account Sign-up dan Login
  2. Research Participant Consent Statement Letter
    3. Consent statement letter document automatic download as a proof of the user approval to be a research participant.
  3. Dashboard
    4. Shows user biodata and user status based on their research form has completed or not.
  4. Biodata Form
  5. Research Questions Form

This application was deployed at Play Store.
Tech to develop this application: Java, XML, Firebase Cloud Firestore.
