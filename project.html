Forum Diskusi

Discussion forum website (2022)

Description

This website is a product of final project in Sanbercode web programming bootcamp. This project is a team project as a chance to learn about git and team collaboration.

Features

  1. Sign-up and Login Page
  2. CRUD User Account
  3. CRUD Questions and Answers

This website was deployed at Heroku.
Website default registered users login data (email - password):
admin@gmail.com - admin, user@gmail.com - user, guest@gmail.com - guest.

Source code and further explanations about this website and project: Gitlab.
Tech to develop this website: Laravel, PHP, Bootstrap, Javascript, MySQL.
